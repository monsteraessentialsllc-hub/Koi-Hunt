KOI HUNT v7 — COLLECTION + PROGRESSION FIX

Updates:
- Collection now scrolls with left/right arrows and shows two world cards at a time.
- Prey color variants now appear randomly; prey score-unlock labels were removed.
- White-Eyed Leucistic Ball Python renamed Blue-Eyed Leucistic Ball Python with bright blue eyes in-game.
- Removed the koi collection text panel from Settings.
- Fixed the collection preview drawing error that left the Rattlesnake and Giant Sea Snake cards as plain blue images.
- Worlds now unlock sequentially: score 100 in the previous world to open the next.
- Added Settings password tools. Enter "monstera" to unlock all levels or reset all levels and scores.
- Completing every world with a best score of 100 reveals the password and unlocks Lunar Koi Mode.

VERSION 9.6 — TOP-DOWN PREY + GOLDEN KOI HOARD FIX
- All prey artwork is kept in a bird's-eye/top-down gameplay perspective.
- Level-one koi now use Kohaku, Showa, and Yamabuki-inspired palettes instead of generic orange/white/gray fish.
- Golden prey in normal gameplay is rarer.
- The home-page Golden Koi now waits roughly 2–4 minutes between appearances.
- Tapping/clicking the home-page Golden Koi reliably starts Dragon's Hoard.
- After losing a temporary Dragon's Hoard run, the player returns to the menu and must wait for another Golden Koi.
