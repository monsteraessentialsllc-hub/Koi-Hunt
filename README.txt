- Green Tree Python world added; chase a frog through moss, branches, leaves, and bugs. Unlocks at 150.
- Rattlesnake Desert world added; chase a lizard through a desert. Unlocks at 220.
- Anaconda Deep Water world added; chase a swimmer instead of a koi. Unlocks at 300.

- Main-menu pond is animated, with the anaconda continuously moving around while hunting the koi.
- Unlock progress uses the highest best score saved in this browser.

CONTROLS:
Arrow keys or WASD to move. Space pauses. Walls wrap. The game ends only when the snake hits itself.
